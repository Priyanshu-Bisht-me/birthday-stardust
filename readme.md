# 🎂 Birthday Card Website

A fun, animated birthday card website built with pure HTML and CSS — no JavaScript!

## Features

- Animated gradient background
- Bouncing "Happy Birthday!" heading
- Floating balloons and twinkling stars
- Wobbling birthday cake image
- Name input field
- Fully responsive (works on mobile)

## CSS Highlights

- `linear-gradient` — colorful animated background
- `@keyframes` — bounce, float, wobble, twinkle animations
- `flexbox` — centered layout
- `transition` + `:focus` — smooth input styling
- `transform` — scale, rotate, translateY
- `box-shadow` + `text-shadow` — depth and glow

## How to Run

1. Download or clone this repo
2. Add a birthday cake image named `cake.jpg` to the folder
3. Open `index.html` in any browser — done!

## Hosting on GitHub Pages

1. Push files to a GitHub repository
2. Go to **Settings → Pages**
3. Set source to `main` branch → Save
4. Live at: `https://yourusername.github.io/repo-name`
